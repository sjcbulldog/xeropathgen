File Menu


Item Description
Insert Waypoint Insert a new waypoint after the current waypoint in the current path. This item is disabled if no path is selected, or no waypoint is seletcted, or if the last waypoint in the path is selected.
Delete Waypoint Delete the currently selected waypoint in the current path. This item is disabled if no path or no waypoint is selected. It is also disabled if the first or last waypoint in a path is selected as these cannot be deleted.
Delete (Path or Path Group) Delte either a path or path group depending on what is selected. The name of this menu item will change based on what is selected.
Add Path Group Add a new path group with a unique name to the path file. The new group will be named "NewGroup" or "NewGroup_number".
Add Path Add a new path with a unique name to the currently selected path group. The new path will be named "NewPath" or "NewGroup_number".
Preferences Display the Preference Editor.
