Preferences Window




This dialog provides a way to edit program preferences.
Name Description
Units The units for the program. The program understands inches, feet, meters, and centimeters.
Output format The output format for trajectory information. This can be JSON or CSV.
NT Server Addr The address of the network table server for path trajectories.
NT Table Name The name of the table to use when publishing trajectories to the network tables.
Grid Enabled If true, a grid is drawn across the field image.
Grid Complete If a grid is drawn, and this is true, the grid is a mesh that extends across the field. If this is false, ticks are drawn along the edge of the field.
Show Equations If true, show the spline equation at the top of the field for each segment of the path. The cursor can be used to move the robot within a path to see the various equations that make up a path.
Equations Stacked If true, and equestions are displayed, this stacks the X and Y equations at the top of the screen. If false, the equestion are displayed side by side at the top of the screen.
