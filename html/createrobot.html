Log File Window




This window is used to create or edit the parameters assocated with a robot.
Name Description
Name The name of the robot. This is the name that will appear in the Robot Menu
Units The units for all speed and distance related measurements.
Effective Length The effecitve length of the robot for purposes of calculating path trajectories. This value is generally identical to the physical length of the robot.
Effective Width The effective width of the robot for the purposes of calculating path trajectories. This value is generally smaller than the physical width of the robot due to the "scrub" of the wheels when turning. Search Chief Delphi for information on calculating the effecitive width (or scrub) of a robot
Physical Length The physical (bumper to bumper) length of the robot. This is used to set the dimensions of the robot as drawn on the field when defining paths.
Physical Width The physical (bumper to bumper) width of the robot. This is used to set the dimensions of the robot as drawn on the field when defining paths.
Max Velocity This is the maximum velocity of the robot for path following. This is not the maximum that the robot can do, but something less than this giving some "headroom" for the software to actually control the robot.
Max Acceleration This is the maximum acceleration of the robot for path following. This is not the maximum that the robot can do, but something less than this giving some "headroom" for the software to actually control the robot.
Max Jerk This is the maximum jerk of the robot for path following. This is not the maximum that the robot can do, but something less than this giving some "headroom" for the software to actually control the robot.
Drive Type This is the drive type. Swerve support is experimental and does not currently work
Time Step This timestep of the control loop for the drive base.
