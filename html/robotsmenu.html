Robots Menu


Item Description
[Robot Name] The top of this menu contains a list of the defined robots. You can change robots by selecting a new robot from the menu.
New Robot Displays the new robot dialog to create a new robot.
Edit Robot Displays the edit robot dialog to edit the properties associated with and existing robot.
Delete Robot Displays the select robot window, allowing the user to select a robot to delete from the program. The current robot can never be deleted.
Export Current Robot Export the currently selected robot to a robot file. This is useful for transferring a robot to another machine.
Import Robot Import a robot from a robot file previously created via the Export Current Robot command.
