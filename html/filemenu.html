File Menu


Item Description
New Creates a new path file. If the current path file has been modified and not saved, you will be prompted to save the current path file.
Open Open an existing path file on disk. If the current path file has been modified and not saved, you will be prompted to save the current path file.
Save Save the current path file to disk. If there is not disk file assocaited with the current path file, for instance if it is new, then you will be prompted for the name of a file to stored the paths in.
Save As Prompt for a new file name and store the current set of paths in the new file. The new file name becomes the name assocaited with the current paths file and subsequence save commands will also save to this new file name.
Close Close the current path file. If the current path file has been modified and not saved, you will be prompted to save the current path file.
Generate Paths As Prompt for a new directory and generate trajectories for each path in this new directory. The output will be either JSON or CSV depending on the output settings option. This can be changed via the Preferences dialog.
Generate Paths Generate trajectories for each path in the path file in the paths directory. If no paths directory is set, you are prompted for a path directory. This can be changed via the Preferences dialog.
Publish Path Publish the generated trajectories for the currently selected path into the network table. By default the table name will be "XeroPlot", but this can be changed via the Preferences dialog. The entry in the table will be PathGroup_PathName_left, PathGroup_PathName_right, and PathGroup_PathName_main for a tank drive. PathGroup will be replaced with the actual name of the path group of the currently selected path. PathName will be replaced with the path name of the currently selected path.
Recent Files This menu will contain up to four path files that have been recently accessed. This is an easy way to find a recently used path file.
