Path Editing Window




The path editing window is used the edit path. When a path is selected in the Path File window, the path is displayed in the path editing window. Here, individual waypoints can be selected with the left mouse button. Selected waypoints can be moved by dragging the waypoint with the left mouse button down. Finally, a waypoint can be rotated by dragging the yellow rotation circle attached to a selected waypoint and moving the mouse around the waypoint. In addition the following keys are useful.
Key Purpose
Up Arrow Moves the waypoint up 12 inches
Shift Up Arrow Moves the waypoint up 1 inch
Down Array Moves the waypoint down 12 inches
Shift Down Arrow Moves the waypoint down 1 inch
Left Arrow Moves the waypoint left 12 inches
Shift Left Arrow Moves the waypoint left 1 inch
Right Arraw Moves the waypoint right 12 inches
Shift Right Array Moves the waypoint right 1 inch
Page Up Rotates the current waypoint counter clockwise 5.0 degrees
Shift Page Up Rotates the current waypoint counter clockwise 0.5 degrees
Page Down Rotates the current waypoint clockwise 5.0 degrees
Shift Page Down Rotates the current waypoint clockwise 0.5 degrees
Insert Inserts a new waypoint after the current waypoint, unless the last waypoint is selected.
Delete Deletes the current waypoint, unless the current waypoint is the first or last waypoint.
